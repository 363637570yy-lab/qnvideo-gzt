# 工作流规范与提示词版本管理

最近更新：2026-06-09

## 功能定位

工作流规范与提示词版本管理用于沉淀 AI 短视频创作平台中的专业生成方案。人物、场景、道具、分镜、剧本、视频和音频都可能有多套专业工作流，操作员页面只做选择，管理员负责配置和维护。

## 工作流类型

| 类型 | 用途 |
| --- | --- |
| 人物生成规范 | 三视图、四视图、角色卡、服装细节、表情表、身份锚点 |
| 场景生成规范 | 多画面场景、俯视图、空间关系、时间天气、风格一致性 |
| 道具生成规范 | 单品图、细节图、使用状态、材质结构、尺寸关系 |
| 分镜生成规范 | 镜头语言、景别、机位、运镜、首尾帧、导演描述 |
| 故事/剧本规范 | 故事结构、分集节奏、对白旁白、小说改编 |
| 图片提示词规范 | 正向词、负面词、参考图说明、画风约束 |
| 视频生成规范 | 首尾帧、动作强度、镜头连续、时长、运动限制 |
| TTS/音频规范 | 语气、语速、角色音色、旁白对白规则 |

## 配置边界

- 全局配置：管理员维护专业规范、默认项、适用模型、适用画风。
- 操作员页面：只选择启用的规范，必要时显示少量安全选项。
- 项目设置：可指定项目默认工作流，但不直接复制整套配置。
- 业务记录：生成任务必须记录当次使用的工作流 ID、版本和渲染后的参数摘要。

## 提示词版本规则

- 任何用于生成的模板都必须有版本号。
- 修改默认工作流不应影响历史任务复现；历史任务保存 `preset_id` 和 `version`。
- 蓝色锁定区、JSON 输出格式、模型硬要求应与可编辑创作提示分层。
- 工作流可被禁用，但禁用后历史任务仍可查看其快照。

## 验收清单

- 管理员能新增、修改、禁用、设默认工作流。
- 操作员只能选择启用工作流，不需要理解复杂参数。
- 历史任务能追溯使用的工作流版本。
- 人物、场景、道具、分镜规范互不混淆。
- 新媒体刷到的新方案能快速录入为新工作流。

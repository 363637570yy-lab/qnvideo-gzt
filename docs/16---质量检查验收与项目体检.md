# 质量检查验收与项目体检

最近更新：2026-06-09

## 功能定位

质量检查与项目体检用于判断一个 AI 短视频项目是否具备继续生成、合成和导出的条件。它把“哪里缺素材、哪里图不可用、哪里视频失败、哪里不一致”变成可见的检查结果。

## 检查对象

- 项目整体设置：画幅、画风、默认语言、默认时长。
- 剧本：分集结构、台词、旁白、可拆镜性。
- 角色：主角/配角身份、主图、参考图、一致性锚点。
- 场景：主场景图、空间关系、时间天气、风格一致性。
- 道具：主图、使用场景、是否影响分镜。
- 分镜：镜头字段、景别、机位、运镜、时长、旁白对白。
- 图片：分镜图、首帧、尾帧、关键帧、锁定状态。
- 视频：每个分镜视频、失败原因、时长、是否可合成。
- 音频：TTS、旁白、对白、缺失项。
- 文件：引用是否存在、是否可访问、是否已被误删。

## 质量状态

建议统一状态：

```text
unchecked       未检查
ready           可用
warning         可继续但有风险
blocked         阻塞后续生成
needs_regen     建议重生成
locked          已锁定，不被批量覆盖
ignored         用户明确忽略
```

## 前端页面规划

- 项目体检面板：整体通过率、阻塞项、警告项、可修复项。
- 分镜体检：缺图、缺视频、缺首尾帧、时长异常、参考图缺失。
- 资产体检：角色/场景/道具主图缺失、一致性风险、未入库。
- 导出前检查：视频、音频、字幕、合成参数是否齐全。
- 一键修复入口：按检查结果触发补图、补视频、补 TTS、重新生成。

## 检查规则示例

- 没有项目画幅：`blocked`。
- 主角没有主图：角色图生成 `blocked`，分镜可 `warning`。
- 分镜没有图片但要图生视频：视频生成 `blocked`。
- 分镜图已锁定：批量生成不得覆盖。
- 视频生成失败但有可重试模型：`needs_regen`。
- 文件引用不存在：`blocked` 并提示清理或重选。
- 视频时长明显偏离分镜时长：`warning`。

## 验收清单

- 用户能清楚知道当前项目为什么不能继续生成或导出。
- 批量操作不会覆盖锁定或已确认的优质结果。
- 缺失素材能被定位到角色、场景、道具、分镜或文件引用。
- 导出前能提示缺视频、缺音频、缺字幕或合成风险。
